# 关于

普通程序员，写过 `Java` 和 `Golang` 代码，略懂一些安全知识

我对开源的看法：要么别开源，开源就别只放个 `README` 和二进制

主要维护的开源项目：

- 一个强大的 `Jar` 文件分析搜索 `GUI` 工具：[jar-analyzer](https://github.com/jar-analyzer/jar-analyzer)
- 基于 `XRAY YAML` 规则的超 **轻量** 引擎：[poc-runner](https://github.com/4ra1n/poc-runner)

其他开源项目：

- 完全不使用第三方库从头开始实现的中文编程语言：[y4-lang](https://github.com/4ra1n/y4-lang)
- 基于 `CPU` 的 `RDRAND` 指令和 `JNI` 实现的随机库：[JRandom](https://github.com/4ra1n/JRandom)
- 可以让你在 `Java` 代码中插入任意的 `JVM` 指令的库：[\_\_asm\_\_](https://github.com/4ra1n/__asm__)
