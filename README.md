# 关于

我主要工作是开发，偶尔研究 `Java` 漏洞，使用的语言主要有 `Golang/Java/Python/C/ASM` 等

这些是开源项目：由于工作繁忙不一定有时间维护更新，欢迎各位师傅们讨论交流以及二开

- [jar-analyzer-gui](https://github.com/4ra1n/jar-analyzer-gui) （一个分析和搜索 `JAR` 文件的 `GUI` 工具）
- [jar-analyzer-cli](https://github.com/4ra1n/jar-analyzer-cli) （根据 `JAR` 文件构建数据库以进行分析和查询）
- [mysql-fake-server](https://github.com/4ra1n/mysql-fake-server) （完全由 `Java` 编写的 `MySQL` 恶意服务端）
- [shell-analyzer](https://github.com/4ra1n/shell-analyzer) （基于 `Java Agent` 的内存马查杀 `GUI` 工具）
- [code-encryptor-plus](https://github.com/Y4Sec-Team/code-encryptor-plus) (使用 `JNI/JVMTI` 加密字节码以保护代码)
- [code-inspector](https://github.com/4ra1n/code-inspector) （基于污点分析的 `Java` 字节码分析工具）
