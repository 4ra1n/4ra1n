# 关于

我主要工作是开发，偶尔研究 `Java` 漏洞，使用的语言主要有 `Java/Golang/C` 等

这里是部分开源项目：由于工作繁忙没有太多时间维护，欢迎各位师傅们讨论交流以及二开

- [code-encryptor-plus](https://github.com/4ra1n/code-encryptor-plus) (使用 `JNI/JVMTI` 加密字节码，支持自定义包名密钥，使用特殊技巧禁止`dump`字节码)
- [jar-analyzer-gui](https://github.com/4ra1n/jar-analyzer-gui) （一个分析和搜索 `JAR` 文件的 `GUI` 工具，提供多种搜索逻辑以及表达式搜索）
- [mysql-fake-server](https://github.com/4ra1n/mysql-fake-server) （完全由 `Java` 编写的 `MySQL` 恶意服务端，用于测试反序列化和文件读取漏洞）
- [shell-analyzer](https://github.com/4ra1n/shell-analyzer) （基于 `Java Agent` 的内存马查杀 `GUI` 工具，实时分析内存马一键清除）
- [code-inspector](https://github.com/4ra1n/code-inspector) （基于污点分析的 `Java` 字节码分析工具，使用模拟 `JVM` 栈帧的技术）
