# 关于

主要维护的开源项目：
- 一个强大的 `Jar` 包分析工具：[jar-analyzer](https://github.com/jar-analyzer/jar-analyzer)
- 一个基于 `Golang` 的中文脚本语言: [y4-lang](https://github.com/4ra1n/y4-lang)

不再维护的开源项目：
- 使用 `JVMTI` 加密解密字节码的工具：[code-encryptor](https://github.com/4ra1n/code-encryptor)
- 基于 `Java Agent` 的内存马分析工具：[shell-analyzer](https://github.com/4ra1n/shell-analyzer)
- 模拟栈帧的 `Java` 字节码分析工具：[code-inspector](https://github.com/4ra1n/code-inspector)
- 老版本 `jar-analyzer` 的 `GUI` 版：[jar-analyzer-gui](https://github.com/4ra1n/jar-analyzer-gui)
- 老版本 `jar-analyzer` 的 `CLI` 版：[jar-analyzer-cli](https://github.com/4ra1n/jar-analyzer-cli)
- 一些 `Java` 层执行 `shellcode` 的例子：[java-gate](https://github.com/4ra1n/java-gate)
- 恶意的 `MySQL` 服务端：[mysql-fake-server](https://github.com/4ra1n/mysql-fake-server)
- 一个 `Java` 内联字节码工具：[\_\_asm\_\_](https://github.com/4ra1n/__asm__)
- 一个 `xray` 扫描器的套壳：[super-xray](https://github.com/4ra1n/super-xray)
